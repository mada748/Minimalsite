Install Guide

Linux and MacOS Only!


Note: The macos guide is not tested yet


clone the repo by writing

git clone https://github.com/mada748/MinimalBrowser

for executing python we need to install and execute venv so do

sudo apt update

and

sudo apt install python3-venv

for executing venv do

python3 -m venv my_env

instead for activating venv do

source my_env/bin/activate

install some py dipencies so do

pip install PyQt5 PyQtWebEngine

and to run the program do

python3 browser.py
